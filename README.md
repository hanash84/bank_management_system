# Banking System

A simple banking system developed in Python using **Object-Oriented Programming (OOP)** and **Inheritance**. The application simulates basic banking operations through a command-line interface.

## Features

- Create a new bank account
- Deposit money
- Withdraw money
- Check account balance
- Input validation
- User-friendly menu

## Concepts Used

- Object-Oriented Programming (OOP)
- Inheritance
- Classes and Objects
- Functions
- Conditional Statements
- Loops

## Technologies

- Python 3

## How to Run

1. Clone the repository.
2. Open the project in Python.
3. Run the main Python file.
